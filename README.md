# 📊 Data Analyst Job Market Analysis with SQL

## Introduction

This project explores the data analyst job market using SQL by analyzing job postings, salaries, and required skills. The goal is to identify which skills are most valuable, both in terms of demand and salary, and to help guide career decisions in data analytics.

---

## Background

The data job market is highly competitive, and knowing which skills to learn can significantly impact career growth. This project answers key questions such as:

- What are the highest-paying data analyst jobs?
- What skills are required for those roles?
- Which skills are most in-demand?
- Which skills offer the highest salaries?
- What are the most optimal skills (high demand + high salary)?

By answering these questions, the project provides actionable insights for aspiring and current data analysts.

---

## 🛠️ Tools I Used

- **SQL (PostgreSQL)** – for querying and analyzing the dataset  
- **Database** – job postings dataset with skills, salaries, and company data  
- **JOINs, CTEs, Aggregations** – to extract meaningful insights  
- **Git & GitHub** – for version control and project sharing  

---

## 📈 The Analysis

### 1. Top Paying Data Analyst Jobs

I identified the top 10 highest-paying remote data analyst jobs with available salary data.

**Key insight:**  
High-paying roles often include senior or director-level positions and are typically fully remote.

---

### 2. Skills Required for Top Paying Jobs

Next, I analyzed which skills are required for those top-paying roles.

**Key insight:**  
Top-paying jobs commonly require:
- SQL
- Python
- Cloud tools (AWS, Azure)
- BI tools (Tableau, Power BI)

---

### 3. Most In-Demand Skills

I calculated which skills appear most frequently in job postings.

**Key insight:**  
The most in-demand skills include:
- SQL
- Excel
- Python
- Tableau / Power BI

These are essential baseline skills for any data analyst.

---

### 4. Skills with Highest Salaries

Then I analyzed which skills are associated with the highest average salaries.

**Key insight:**  
Higher-paying skills tend to include:
- Big data tools (Spark, Kafka)
- Cloud platforms (AWS, GCP)
- Machine learning frameworks (TensorFlow, PyTorch)

---

### 5. Most Optimal Skills (High Demand + High Salary)

Finally, I combined demand and salary data to identify the most optimal skills.

**Key insight:**  
The best skills to learn are those that:
- Appear frequently in job postings  
- Offer high average salaries  

Examples:
- SQL
- Python
- Cloud technologies
- Data engineering tools

---

## 📚 What I Learned

Through this project, I learned:

- How to use **CTEs (Common Table Expressions)** to structure complex queries  
- How to combine datasets using **JOINs**  
- How to perform **data aggregation and filtering**  
- How to translate business questions into SQL queries  
- How to analyze both **demand (COUNT)** and **value (AVG salary)**  

---

## ✅ Conclusions

- **SQL is the most essential skill** for data analysts  
- **Python and cloud tools** significantly increase salary potential  
- **BI tools** are widely required across most roles  
- The best strategy is to focus on skills that are both:
  - High demand  
  - High paying  

💡 **Final takeaway:**  
To maximize career opportunities as a data analyst, focus on a combination of:
- SQL + Python  
- Visualization tools (Tableau / Power BI)  
- Cloud & big data technologies
